## Projects

### 1. Flipkart Sales Analysis Dashboard
Analyzed 10M+ e-commerce sales records for trends and insights.
- Key Features: Payment methods (UPI, COD, Debit/Credit Cards), Regional sales (North, South, East, West, Central), Top categories (Grocery, Electronics, Fashion), Sales trends over time, Rating performance.
- Built with: Power BI DAX measures (CALCULATE, SUMX etc.), Power Query for data cleaning.
